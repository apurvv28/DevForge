# Security Policy

## Supported Versions

| Version | Supported |
| --- | --- |
| 1.x | Yes |

## Reporting a Vulnerability

Report security issues through GitHub Security Advisories using private disclosure.

Do not open a public issue for a suspected vulnerability.

## Response Timeline

- Acknowledge receipt within 48 hours.
- Provide a patch or mitigation for critical issues within 14 days.
- Triage and prioritize lower-severity issues as part of the normal release cycle.
